# Day-3 HTML Practice

## Project Description

This is my **Day-3 HTML practice project**.

আজকের প্রজেক্টে আমি একটি simple **Learning Platform Homepage** তৈরি করেছি যেখানে আমি semantic layout, block vs inline elements, class & id logic, এবং media embedding practice করেছি।

The main goal was not design —
The goal was to understand **structure, grouping, and proper HTML usage**.

---

## Objective

The objective of this task was to:

- Practice semantic HTML structure
- Understand block vs inline elements
- Use `<div>` properly as a container
- Learn the difference between `id` and `class`
- Embed video, audio, and YouTube content
- Use basic inline styles
- Add a simple `<script>` tag

এই প্রজেক্টের মাধ্যমে আমি HTML কে আরও logically চিন্তা করতে শিখেছি।

---

## Page Structure (Semantic Layout)

This page is divided using proper semantic tags:

- `<header>` → Website name and tagline
- `<nav>` → Navigation menu
- `<section>` → Main learning content
- `<footer>` → Copyright & contact

I avoided using only `<div>` and focused on meaningful structure.

This helped me understand that HTML is about **meaning**, not just layout.

---

## Block vs Inline Elements

Inside the main section:

### Block Elements Used:

- `<div>`
- `<h2>`
- `<p>`

### Inline Elements Used:

- `<span>`
- `<a>`
- `<b>`

From this section, I learned:

- Block elements take full width.
- Inline elements stay inside text flow.
- Structure becomes clearer when used properly.

---

## Div as Container

I created two course boxes using `<div>`.

Each course box contains:

- A heading
- A paragraph
- A link

Both course boxes share the same:

```html
class="course-box"
```

This shows reusability in HTML.

---

## ID vs Class Logic

- `id="main-learning"` → Used only once (unique section)
- `class="course-box"` → Used multiple times (reusable component)

Here I understood:

- **id = unique**
- **class = reusable**

This is very important for future CSS and JavaScript learning.
