# Contact Form – HTML Beginner Project

## Project Description

This is my **first HTML practice project**.
I created a **simple Contact Form** to understand how basic HTML works and how forms collect user input on a web page.

---

## What Are We Building?

We are building a **simple Contact Form** that contains:

- A main heading: **Contact Us**
- A short paragraph introducing the form
- Input fields:
  - **Name** (text input)
  - **Email** (email input)
  - **Message** (textarea for multi-line text)

- A **Submit button**
- One **HTML comment**

---

## What Is the HTML Comment Used For?

In this project, an HTML comment is used like this:

```html
<!-- This input is used to collect the user's email address -->
```

- HTML comments are **not visible** in the browser
- They are used to **explain code**
- They help beginners and other developers understand **what a part of the code does**
