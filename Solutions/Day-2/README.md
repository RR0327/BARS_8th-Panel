এখানে আমি শিখেছি HTML কিভাবে একটি page এর structure তৈরি করে, calculation বা logic নয়।

## What Is Included in This Page?

### 1. Basic HTML Structure

This page includes a proper HTML boilerplate with head and body sections।
এখানে `<title>` এবং favicon ব্যবহার করে browser-level basics শেখা হয়েছে।

---

### 2. Text & Formatting

This section contains headings, paragraphs, line breaks, and a horizontal line।
এছাড়াও basic formatting tags যেমন bold, italic, এবং underline ব্যবহার করা হয়েছে।

এই অংশ থেকে বোঝা যায় content কিভাবে readable করা হয়।

---

### 3. Lists, Links & Image

The page includes both ordered and unordered lists to represent information clearly।
একটি clickable link এবং একটি image ব্যবহার করা হয়েছে proper `alt` text সহ।

এখানে accessibility আর structure এর importance বোঝা যায়।

---

### 4. Form

A simple login form is included with username and password fields।
Labels ব্যবহার করা হয়েছে যাতে form টা user-friendly হয়।

This helped me understand how HTML collects user input.

---

### 5. Semantic HTML

Semantic tags are used to give meaning to content, not just visual style।
Important text, emphasis, highlighted words, price updates, এবং scientific formulas—সব কিছু semantic way তে লেখা হয়েছে।

এই অংশটা HTML শেখার সবচেয়ে গুরুত্বপূর্ণ অংশ।

---

### 6. Quotes & Citations

Both long and short quotations are included using proper tags।
Abbreviation এবং contact information দেখানোর জন্য semantic tags ব্যবহার করা হয়েছে।

এতে content আরও meaningful এবং professional হয়।

---

### 7. Product Price Table

A product price table is created to display structured data।
Old price, new price, এবং update status clearly table এর মাধ্যমে দেখানো হয়েছে।

এই অংশ থেকে বুঝেছি table কেন real projects এ খুব common।
