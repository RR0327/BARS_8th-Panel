# Your 1st Day Challenge:

Create a **simple contact form** that includes:

- A main heading (`<h1>`) for "Contact Us".

- A paragraph (`<p>`) introducing the form.

- Input fields for "Name" (`text`), "Email" (`email`), and "Message" (a `<textarea>` for multi-line text).

- A submit button (`<input type="submit">`).

- Add a comment to explain the purpose of one of your input fields.

> When you finish, send your HTML code for review "Provide the Github link not github live display link".
