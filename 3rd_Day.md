### Day-3 HTML Practice Task

**Task Title:**
Single-Page Learning Platform Layout

---

## Objective

Create **one HTML page** that represents a small learning platform homepage.

---

## Task Description

You are building a **simple educational website homepage** that has:

- A clear page structure
- Reusable sections
- Embedded media
- Proper use of class and id
- Visible difference between block and inline elements

---

## Task Requirements

### 1. Page Structure (Semantic Layout)

Your page **must be divided using semantic layout elements**, not only `<div>`:

- `<header>` → Website name and short tagline
- `<nav>` → Navigation menu (3 links)
- `<section>` → Main learning content
- `<footer>` → Copyright + contact info

Each section should contain **meaningful content**, not empty text.

---

### 2. Block vs Inline Demonstration

Inside the **main section**:

- Use at least **three block elements** (`<div>`, `<h2>`, `<p>`)
- Inside one paragraph, use **multiple inline elements**:
  - `<span>`
  - `<a>`
  - `<b>`

The layout should clearly show how block and inline elements behave differently.

---

### 3. Div as a Container

Inside the `<section>`:

- Create **two course boxes** using `<div>`
- Each box must contain:
  - A heading
  - A paragraph
  - A link

These `<div>` elements should act as **grouped containers**, not random wrappers.

---

### 4. Class and ID Logic

- Assign **one unique `id`** to the main section (used only once).
- Assign **one shared `class`** to both course boxes.

You should be able to explain:

- Why `id` is used only once
- Why `class` is reused

---

### 5. Media Content

Inside the page:

- Add **one video** using `<video controls>`
- Add **one audio** using `<audio controls>`

Use placeholder file names if needed.

---

### 6. Iframe (YouTube)

- Embed **one YouTube video** using `<iframe>`
- Place it logically inside the learning section (not randomly)

---

### 7. HTML Styles and Script (Basic Use)

- Use the `style` attribute on **at least two elements** to change:
  - text color OR
  - background color OR
  - font size

- Add a `<script>` tag that:
  - Writes a message in the browser console
    OR
  - Shows a simple alert message

---

## Thinking Hint

Before writing code, ask yourself:

- Which parts are **unique**?
- Which parts are **reusable**?
- Which elements should **group content**?
- Where does media logically belong?

If someone understands today’s topics, they will understand **what to build and how to build it**.
